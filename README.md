# PR Station
PR Station is a press release site created through Wordpress by way of SMACSS (http://smacss.com) and Javascript ES6 modular methodology. It has been released and caters to hundreds of users daily.

#### Live site
https://prstation.ph/

#### Installation and Usage
Open up your terminal and move to the folder where you cloned the project.<br>
Run *npm i* to install all necessary modules and dependencies.<br>
Then, *npm run dev* to compile all assets and build files.<br>
Lastly, *npm run watch-poll* to watch all relevant files for changes. Webpack will then automatically recompile your assets when it detects a change.

#### Dependencies
jQuery ^3.4.1 <br>
laravel-mix
